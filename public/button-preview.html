New "Register Token" Button Added:
Features:
  • Green Button: Eye-catching success color
  • Notification Icon: Clear visual indicator
  • Responsive Text: Shows "Register Token" on desktop, "Token" on mobile
  • API Integration: Calls /api/save-token endpoint
  • Platform Detection: Works with both mobile (Capacitor) and web
  • Loading States: Shows spinner while processing
  • Success Feedback: SweetAlert notifications and visual feedback
How it works:
  1. Mobile (Capacitor): Requests push permission → Gets FCM token → Saves to database
  2. Web Browser: Shows informational message about mobile-first approach
  3. Success: Button changes to "Token Registered" with checkmark and reloads page
  4. Error Handling: Shows detailed error messages via SweetAlert
